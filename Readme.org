* Technology stack
** Go
** SQLite 3
* Features
** List Pens
** Add Pens
** Hard Coded Pen characteristics
** Hard coded Nord theme
** The SQLite database is created as follows,

#+begin_src
CREATE TABLE pens (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT,
    maker TEXT,
    color TEXT,
    material TEXT,
    nib_size TEXT,
    nib_color TEXT,
    filling_system TEXT,
    trims TEXT,
    year INTEGER,
    price REAL,
    misc TEXT
);
#+end_src
