* Technology stack
Currently using
- Go
- SQLite 3
* Features
- List Pens
- Add Pens
- Modify pens
- Hard Coded Pen characteristics
- Hard coded Nord theme or  bug
- Can import from and export to a CSV
- Sorting by columns is supported

** Go Modules required
- Following modules are needed

#+begin_src
go get github.com/mattn/go-sqlite3
#+end_src

** Database structure
- The SQLite database is created as follows,

#+begin_src
CREATE TABLE pens (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT,
    maker TEXT,
    color TEXT,
    material TEXT,
    nib_size TEXT,
    nib_color TEXT,
    filling_system TEXT,
    trims TEXT,
    year INTEGER,
    price REAL,
    misc TEXT
);
#+end_src

** To run the code

#+begin_src
go run main.go
#+end_src

In case you are unable to connect to the database, run the following and then the run command:

#+begin_src
go clean
#+end_src
