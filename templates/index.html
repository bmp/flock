Flock: Personal Fountain Pen Database

Welcome to Flock, this is a personal fountain pen database system. Currently, it has the following features,

  1. Maintain a list of fountain pens.
  2. Export and import from CSV files, there is no lock-in whatsoever for you. You can download the CSV file and use any text editor of your choice to continue to maintain your inventory of fountain pens.
  3. There are absolutely no social features in this inventory system and it shall remain so.
  4. This software has very limited JavaScript (mainly for confirmation dialogues, date pickers, table sorters, and such).
